# MAX Admin MVP — v27 CRM Leads Board

Версия v27 добавляет полноценный раздел **CRM / Заявки из мини-приложения** поверх рабочей версии v26.

## Что есть в v27

- CRM-доска заявок из `max_miniapp_leads`
- карточки заявок с именем, телефоном, адресом, размерами, фото, источником и предварительной суммой
- статусы: новая, в работе, замер назначен, расчёт отправлен, заказ, отказ, дубль
- поиск по телефону, имени, адресу и кампании
- фильтр по статусу и UTM-кампании
- заметка менеджера и следующее действие
- быстрые кнопки WhatsApp и звонка
- KPI сверху: всего заявок, новые, в работе/замер, заказы, предварительная сумма
- цены в mini app по-прежнему скрыты от клиента
- Vercel Hobby совместимость сохранена: одна функция `api/[...route].js`

## Проверка версии

После деплоя откройте:

```text
https://adminka-max.vercel.app/api/version
```

Ожидаемо:

```json
{
  "ok": true,
  "version": "v27-crm-leads-board"
}
```

## Supabase

Если v22/v23/v26 уже применялись, основной набор таблиц уже есть. Для заметок и дополнительных CRM-полей выполните SQL:

```text
supabase/hotfix-crm-leads-board.sql
```

Он добавляет поля:

```text
crm_note
crm_next_action
crm_updated_at
crm_updated_by
```

## Важно

В папке `api` должен быть только один файл:

```text
api/[...route].js
```

В корне проекта не нужен `vercel.json` для Hobby-версии.
