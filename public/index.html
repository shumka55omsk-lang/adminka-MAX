Интеграция

Webhook MAX

Автоматическое получение chat_id, диагностика API и управление подписками.

После подключения Webhook бот будет сам сохранять группы в Supabase, когда его добавили в группу или в группе появилось событие.

В Vercel укажи PUBLIC_BASE_URL=https://твой-домен.vercel.app и MAX_WEBHOOK_SECRET. Endpoint: /api/max-webhook.

Группы

Группы MAX

Список групп для рассылок, ручное добавление и управление chat_id.

Введите пароль и нажмите “Войти / загрузить группы”.

Если Supabase не подключён, список групп берётся из переменной MAX_GROUPS, а добавление групп из админки недоступно.

CTA и ссылки

Кнопки под постом

Настрой ссылки для WhatsApp, Telegram, сайта и кнопки расчёта стоимости.

Ссылка мини-приложения будет сформирована автоматически.

MAX принимает в кнопках только ссылки формата http:// или https://. Телефон лучше писать прямо в тексте поста: 48-98-78.

Отправка

Публикация поста

Когда всё готово — отправьте пост сразу в выбранные группы.

Автоматизация

Расписание отправки

Планируйте публикации заранее и запускайте очередь вручную при необходимости.

Выбери дату и время, нажми “Запланировать”. Отправка произойдёт автоматически через Vercel Cron или внешний cron-сервис. Админку держать открытой не нужно.

Запланированных постов пока нет.

Время вводится по вашему устройству. Cron проверяет очередь по UTC, но дата сохраняется правильно с учётом вашего часового пояса.
